## 일치율 계산 예시

항목 9개 중 ✅ 8개, ⚠️ 1개인 경우:

```
일치율 = ((8 + 1 * 0.5) / 9) * 100
       = (8.5 / 9) * 100
       = 94.4%
       ≈ 94%

→ 평가 등급: Good (일부 보완 필요)
```

항목 6개 중 ✅ 6개인 경우:
```
일치율 = (6 / 6) * 100 = 100%
→ 평가 등급: Perfect
```
