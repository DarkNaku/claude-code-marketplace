# 장르별 밸런스 기준값

레퍼런스 게임 역설계와 장르 관례 기반의 초기 수치 참고 데이터.

> 모든 수치는 추정값 또는 역설계 추정치입니다. 플레이테스트를 통해 반드시 조정합니다.

---

## 로그라이크 서바이벌 (Vampire Survivors류)

### 핵심 수치 패턴

| 파라미터 | 일반적 범위 | 비고 |
|---------|-----------|------|
| 세션 목표 길이 | 10~30분 | 장르 표준 |
| 첫 레벨업까지 | 10~30초 | 빠른 첫 성취 제공 |
| 레벨업 간격 (초반) | 10~20초 | 초반 빠른 성장감 |
| 레벨업 간격 (후반) | 30~60초 | 후반 안정화 |
| 최대 레벨 | 30~100 | 게임 복잡도에 따라 |
| 플레이어 기본 HP | 80~150 | 적 공격력 대비 10~20회 생존 |
| 피격 무적 시간 | 0.5~1.5초 | 연속 피격 방지 |
| 기본 이동 속도 | 3~5 units/sec | |
| 초기 적 스폰 간격 | 1~3초 | |
| 초기 동시 최대 적 수 | 5~20마리 | |
| 후반 동시 최대 적 수 | 100~500+ | 화면을 가득 채우는 연출 |

### 경험치 곡선 패턴 (Vampire Survivors 추정)

```
레벨 1→2:  경험치 5   (적 1~2마리)
레벨 2→3:  경험치 10  (적 2~3마리)
레벨 N→N+1: 이전 레벨의 약 1.2~1.5배
```

**공식**: `EXP(L) = 5 × 1.3^(L-1)` (근사값)

### 스폰율 시간 곡선

```
0~2분:   1마리/2초,  최대 20마리
2~5분:   1마리/1초,  최대 50마리
5~10분:  1마리/0.5초, 최대 150마리
10분+:   점진적 증가, 보스 등장 주기 추가
```

|---------|-----------|------|

| 레벨당 이동 수 | 15~35 | 초반 넉넉, 후반 빡빡 |
| 레벨 클리어 소요 시간 | 1~5분 | |
| 레벨당 평균 시도 횟수 (설계 목표) | 1.5~3회 | |
| 첫 50레벨 평균 시도 | 1~1.5회 | 온보딩: 쉬움 |
| 50~200레벨 평균 시도 | 2~3회 | 중반: 적당한 도전 |
| 200레벨+ 평균 시도 | 3~5회 | 후반: 난이도 상승 |
| 생명(Heart) 최대 보유 | 5개 | |
| 생명 재생 시간 | 30분/개 | |

### Near Miss 설계 (이탈 방지)

```
목표: 실패한 시도 중 ~30%는 이동 수 2~3개 이하 남기고 실패
→ 이를 위해 레벨 목표를 "이동 수 기본 계획의 약 90~95%"로 설정
→ 보드 배치에서 목표 달성 경로를 1~2개만 허용
```

### 핵심 수치 패턴

| 파라미터 | 일반적 범위 | 비고 |
|---------|-----------|------|
| 보드 크기 | 5×8 ~ 5×9 | |
| 에너지 최대 보유량 | 20~30개 | |
| 에너지 재생 속도 | 분당 1개 | 30분 만충전 |
| 오브젝트 소환 비용 | 1~5에너지 | 레벨에 따라 |
| 한 세션 평균 플레이 시간 | 10~20분 | 에너지 소진까지 |
| 일일 접속 횟수 (설계 목표) | 3~5회 | |
| 미션 1개 완료 소요 세션 | 1~2세션 | |
| 공간 1개 복원 소요 시간 | 수시간~1일 | |

### 병합 트리 깊이

```
일반적으로 레벨 7~10 깊이
Level 1 → 2 → 3 → ... → 7 (목표 오브젝트)

소환 비용 = 1에너지 (Level 1 기준)
Level N 오브젝트를 만들기 위한 에너지:
  Level 1: 1에너지
  Level 2: 2에너지 (1+1)
  Level 3: 4에너지 (2+2)
  Level N: 2^(N-1) 에너지
```

## 타워 디펜스

### 핵심 수치 패턴

| 파라미터 | 일반적 범위 | 비고 |
|---------|-----------|------|
| 웨이브당 소요 시간 | 30~90초 | |
| 배치 페이즈 시간 | 10~30초 | |
| 첫 웨이브 적 수 | 10~20마리 | |
| 웨이브당 적 증가율 | 1.2~1.5배 | |
| 기지 HP | 20~100 (정수) | 작을수록 긴장감 |
| 기본 타워 비용 | 50~100골드 | |
| 첫 타워 구매까지 소요 웨이브 | 0.5~1 웨이브 | 즉시 배치 가능 |
| 기본 타워 DPS | 적 HP/5초 (1층 기준) | 5타에 처치 |
| 타워 사거리 | 맵 너비의 15~25% | |

### 골드 수급 균형

```
웨이브 1 적 10마리 처치 → 100골드
기본 타워 1개 = 50골드
→ 웨이브 1 완료 후 타워 2개 배치 가능

이 비율로 진행 속도 조절:
  - 더 빠른 진행: 적 보상 ↑ 또는 타워 가격 ↓
  - 더 전략적: 적 보상 ↓ 또는 타워 가격 ↑
```

## 공통 수치 설계 원칙

### "10배 규칙" — 수치 범위 검증

게임의 가장 약한 상태와 가장 강한 상태의 수치 차이:

```
초기 상태 수치 × 10 ≈ 최종 상태 수치  (일반적인 범위)

예: 초기 공격력 6 → 최대 공격력 60~100
예: 초기 적 HP 30 → 최종 적 HP 300~500
```

이 범위를 크게 벗어나면:
- 10배 미만: 성장감 부족, 후반 지루함
- 100배 이상: 수치 인플레이션, 가독성 저하

### 첫 성공까지의 시간 기준

| 장르 | 첫 성공 경험 | 목표 시간 |
|------|-----------|--------|
| 서바이벌 로그라이크 | 첫 레벨업 | 10~30초 |
| 덱빌딩 | 첫 전투 승리 | 1~3분 |
| 3매치 퍼즐 | 첫 레벨 클리어 | 30~90초 |
| 머지 | 첫 병합 + 새 오브젝트 | 10~20초 |
| 타워 디펜스 | 첫 웨이브 방어 성공 | 1~2분 |

### 플레이어 허용 실패 횟수 (MVP 기준)

초보 플레이어가 핵심 루프를 이해하기 전까지 허용되는 최대 게임 오버 횟수:
- 캐주얼 게임: 0~2회 (실패 없이 학습)
- 중간 난이도: 3~5회
- 코어 게임: 10회+ (반복 실패가 학습 과정)

### 초기 파라미터 출발점 공식

아무 데이터도 없을 때 최초 수치 설정:

```
플레이어 HP = 피격 횟수 × 적 공격력
  → "몇 번 맞으면 죽는가"를 먼저 결정

적 HP = 플레이어 공격력 × 처치 타격 수
  → "몇 번 때리면 죽는가"를 먼저 결정

경험치 = 세션_길이 / 목표_레벨업_횟수 / 마이크로루프_횟수
  → 역산으로 도출
```
